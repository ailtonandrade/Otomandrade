### <div align="center">iControl Rastreamento Veicular</div>

<div align="center">Landing page para a empresa iControl, especializada em rastreamento veicular via satélite. Oferecemos soluções para frotas empresariais e veículos particulares, com mensalidades a partir de R$ 35 e adesão de R$ 300.</div>

## Sobre o Projeto

Este é o site landing page da iControl Rastreamento, desenvolvido com Bootstrap para uma apresentação profissional e responsiva. O site destaca nossos serviços de rastreamento para carros, motos e motos elétricas, com foco em segurança e praticidade.

### Funcionalidades
- Design responsivo e moderno
- Seções para Sobre, Serviços e Contato
- Integração com WhatsApp e Instagram
- Cores chamativas: azul, laranja e cinza

### Tecnologias Utilizadas
- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (jQuery, Font Awesome)

### Como Usar
1. Clone o repositório
2. Abra o `index.html` no navegador
3. Personalize conforme necessário
